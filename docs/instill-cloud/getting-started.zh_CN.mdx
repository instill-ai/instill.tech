## Instill Cloud is in **Open Alpha**

This documentation will guide you through the process of getting started with Instill Cloud, a powerful platform that serves as the foundation for AI-first applications.
During the **Open Alpha** phase, all features are available for **FREE**. For more information, see the [Pricing](/pricing) page.
