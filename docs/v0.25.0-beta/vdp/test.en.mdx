When building a pipeline, you can effortlessly test it with a single click, observing the output at each stage in real-time.

<ZoomableImg
  src="https://github.com/instill-ai/community/raw/main/hacktoberfest2023/assets/test-pipeline.gif"
  alt="Visually test pipelines with a single click, viewing output at each stage"
/>

On the Console pipeline builder, press the **Save** and **Test** button to save and test your pipeline.

<ZoomableImg
  src="https://github.com/instill-ai/community/raw/main/hacktoberfest2023/assets/test-pipeline-1.png"
  alt="Test a pipeline on pipeline builder"
/>

Fill in the required input data and click **Run** to visualize the pipeline triggering result.

<ZoomableImg
  src="https://github.com/instill-ai/community/raw/main/hacktoberfest2023/assets/test-pipeline-2.png"
  alt="Visually test the pipeline by filling the required inputs"
/>

<ZoomableImg
  src="https://github.com/instill-ai/community/raw/main/hacktoberfest2023/assets/test-pipeline-3.png"
  alt="Pipeline run result visualization example"
/>
