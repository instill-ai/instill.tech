The Base64 Operator enables users to encode or decode a string in Base64 format.

## Release Stage

`Alpha`

## Operator Configuration

The operator configuration is used for setting up the input data and parameters of this component. The configuration is configured in pipeline recipe, please refer to [pipeline](../../core/concepts/pipeline) for more details.

### Encode

| Input  | Type   | Note                      |
| :----- | ------ | ------------------------- |
| task\* | string | TASK_ENCODE               |
| data\* | string | String data to be encoded |

| Output | Type   | Description           |
| :----- | ------ | --------------------- |
| data   | string | Encoded Base64 string |

### Decode

| Input  | Type   | Note                        |
| :----- | ------ | --------------------------- |
| task\* | string | TASK_DECODE                 |
| data\* | string | Base64 string to be decoded |

| Output | Type   | Description    |
| :----- | ------ | -------------- |
| data   | string | Decoded string |

## Example

Example input and output data for each task:

<CH.Code>

```json TASK_ENCODE
// Input
{
  "task": "TASK_MARSHAL",
  "data": "I love VDP"
}
// Output
{
  "data": "SSBsb3ZlIFZEUA=="
}
```

```json TASK_DECODE
// Input
{
  "task": "TASK_MARSHAL",
  "data": "SSBsb3ZlIFZEUA=="
}
// Output
{
  "data": "I love VDP"
}
```

</ CH.Code>
