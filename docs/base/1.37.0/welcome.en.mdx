## Base: Essential Services for VDP and Model

[**Base**](https://github.com/instill-ai/base), also referred to as **Instill Base**, is a fundamental component that hosts essential services, including user management servers, databases, and third-party observability tools.
Base lays the groundwork for launching VDP and Model seamlessly.
Additionally, it incorporates [Console](https://github.com/instill-ai/console), a user-friendly web-based UI application that enhances the accessibility of VDP and Model.
