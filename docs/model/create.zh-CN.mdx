**Instill Model** or simply Model, an advanced and scalable ModelOps/LLMOps platform focused on empowering users to seamlessly import, serve, fine-tune, and monitor Machine Learning (ML) models for continuous optimization.

## Create a model namespace

1. Start by clicking on `Create A Model` on your model page.
2. **Configure Settings**
    - **Owner**: Select whether the model will be under your personal account or your organization's account.
    - **Model ID**: Enter a name for your model namespace.
    - **Description**: Optionally, add a description to give more context about the model.
    - **Visibility**: Choose if you want your model to be public or private.
    - **Model Task**: Select the kind of [AI Tasks](ai-task) this model will perform.
    - **Cloud Region**: Decide which cloud provider and which region the model will be served on.
    - **Hardware**: Decide what kind of hardware type this model will run on, cost will vary.

<InfoBlock type="info" title="info">
  A public model is viewable and triggerable by all Instill users
</InfoBlock>

3. Now you have created an empty model namespace, the next step is to [prepare](prepare) your custom models and start serving!
