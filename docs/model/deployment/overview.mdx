Versatile Data Pipeline (VDP) provides AI and data practitioners an infrastructure to build up complex pipelines to
extract unstructured data from data connectors and process it with AI connectors.

VDP currently supports deployment via:

- [Docker Compose](/docs/vdp/deployment/docker-compose)
- [Kubernetes using Helm](/docs/vdp/deployment/kubernetes-using-helm)
