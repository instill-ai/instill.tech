The OpenAI AI connector enables you to connect the AI models served on the OpenAI Platform for the **Text to Image** AI task.

## Release stage

`Alpha`

## Configuration

The up-to-date configuration is maintained [here](https://github.com/instill-ai/connector-ai/blob/main/pkg/openai/config/definitions.json).

| Field     | Type   | Note                                                                                 |
| :-------- | :----- | :----------------------------------------------------------------------------------- |
| api_key\* | string | Enter your OpenAI API key. To find your keys, go to your DreamStudio's Account page. |
| task\*    | string | AI task type. Allowed tasks include "Text to Image".                                 |
| engine\*  | string | OpenAI Engine (model) to be used. The default is "stable-diffusion-xl-beta-v2-2-2".  |

### No-code setup

<InfoBlock type="info" title="Prerequisites">

To create a OpenAI AI connector, you'll first need to create a
OpenAI account by visiting [dreamstudio.ai](https://dreamstudio.ai).
Access to your API keys can then be managed through DreamStudio's Account
page. All new users are granted 25 free credits upon creating an account.

ðŸ‘‰ Alternatively, you can try out [Instill Cloud](https://console.instill.tech)
to use our pre-created OpenAI AI connector for FREE during Open Alpha.

</InfoBlock>

To create a OpenAI AI connector:

1. Go to the **AI** page and click **Set up new AI**
2. In the **Set Up New AI Connector** page, fill an ID for your model, this will be the unique identifier of this connector
3. [Optional] Give a short description of your connector in the **Description** field
4. Click the **AI Connector Type** â–¾ drop-down and choose **OpenAI**
5. Fill the following information and click **Set up**

   - Fill **API Key**. Follow the OpenAI [documentation](https://platform.stability.ai/docs/getting-started/authentication) to find your API key
   - Click **Task** â–¾ drop-down and choose **Text to Image**
   - Click **Engine** â–¾ drop-down and choose the AI model you want to use. check [here](https://platform.stability.ai/docs/features/api-parameters#engine) to get the model details

6. By default, the connector is disconnected. To connect and use AI connector, go to the connector page and click **Connect**.

### Low-code setup

1. Create a OpenAI AI connector

```shellscript cURL
curl -X POST http://localhost:8080/v1alpha/connectors -d '{
  "id": "my-stable-diffusion-xl-1024-v0-9",
  "connector_definition_name": "connector-definitions/ai-stability-ai",
  "configuration": {
    "api_key": "<stability-ai-api-key>",
    "task": "Text to Image",
    "engine": "stable-diffusion-xl-1024-v0-9"
  }
}'
```

2. Connect the connector `my-stable-diffusion-xl-1024-v0-9`

```shellscript cURL
curl -X POST http://localhost:8080/v1alpha/connectors/my-stable-diffusion-xl-1024-v0-9/connect
```

For other operations, please refer to the [VDP Protobufs](https://buf.build/instill-ai/protobufs).
