[**Start**](start) enables users to define data formats customized for their AI-first applications.

## Release stage

`Alpha`

## Configuration

| Field | Type | Note                                                                                                 |
| :---- | :--- | :--------------------------------------------------------------------------------------------------- |
| N/A   | N/A  | This operator doesn't take any configuration so the configuration field is an empty JSON object `{}` |

### No-code setup

To create a Start operator:

1. Go to the **Operator** page and click **Set up new operator**
2. Click the **Operator type** â–¾ drop-down and choose **Start**

Now go to the **Operator** page, the corresponding operator with ID `start` should be connected.

### Low-code setup

To create a Start operator:

```shellscript cURL
curl -X POST http://localhost:8080/v1alpha/operators -d '{
  "id": "start",
  "operator_definition_name": "operator-definitions/start",
  "configuration": {}
}'
```

For other operations, please refer to the [VDP Protobufs](https://buf.build/instill-ai/protobufs).
