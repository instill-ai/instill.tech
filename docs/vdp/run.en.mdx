You can either execute your own pipeline on Pipeline Builder or operate another's pipeline through Pipeline Preview Page

<ZoomableImg
  src="/docs-assets/vdp/run-our-own-pipeline.gif"
  alt="Run a Pipeline on builder"
/>

To run another user's pipeline through the Pipeline Preview page on Instill Hub, follow these steps:

1. Navigate to Instill Hub and locate a pipeline that interests you.
2. Click on the title of the pipeline to access the Pipeline Preview Page.
3. Read the instructions or examples provided in the `Pipeline Description` section to understand how to use it.
4. Provide the necessary inputs or upload the required files to configure the pipeline.
5. Click the `Run` button to execute the pipeline and view the results.
